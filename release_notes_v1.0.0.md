# Code-to-Spec Roofing™ Standard v1.0.0

Initial public release of the Code-to-Spec Roofing™ white paper and Inspector Roofing definition standard.

This release includes:

- Formal white paper in PDF, DOCX, and Markdown
- State-by-state residential code adoption matrix
- Source register with official and industry references
- Citation metadata (`CITATION.cff`)
- Zenodo metadata (`.zenodo.json`)
- DOI workflow instructions

Code-to-Spec Roofing™ defines a roof installation as compliant only when it aligns with adopted code, local AHJ amendments, manufacturer installation instructions, warranty eligibility requirements, safety obligations, system performance, and a verifiable closeout record.

Educational notice: This release is not legal advice, engineering advice, public adjusting advice, or insurance coverage advice. Always verify current code and amendments with the local authority having jurisdiction.
