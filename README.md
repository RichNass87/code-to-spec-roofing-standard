# Code-to-Spec Roofing™ Standard

**Code-to-Spec Roofing™** is an Inspector Roofing definition standard and research white paper about the intersection of state/local building code adoption, manufacturer roofing specifications, warranty eligibility, AHJ enforcement, and installation documentation.

- `paper/code_to_spec_roofing_white_paper.md`

- `docs/source_register.csv`

## Core definition

Code-to-Spec Roofing™ means roofing installation is treated as compliant only when it satisfies the adopted code, local AHJ amendments, manufacturer installation instructions, warranty eligibility requirements, safety obligations, system performance, and a verifiable closeout record.
